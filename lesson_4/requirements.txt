Отчет по негативным проверкам для методов класса StringUtils:

1. Метод capitalize
Проверка с пробелом в начале строки:
Вход: " skypro"
Ожидаемый результат: " skypro"
Результат: Пробел не изменился, строка не изменилась.

Проверка с пустой строкой:
Вход: ""
Ожидаемый результат: ""
Результат: Пустая строка, никаких изменений.

Проверка со строкой, состоящей только из пробелов:
Вход: " "
Ожидаемый результат: " "
Результат: Пробелы не изменились.

2. Метод trim
Проверка с пустой строкой:
Вход: ""
Ожидаемый результат: ""
Результат: Строка остается пустой.

Проверка со строкой, состоящей только из пробелов:
Вход: " "
Ожидаемый результат: ""
Результат: Все пробелы удалены, строка пустая.

3. Метод to_list
Проверка с пустой строкой:
Вход: ""
Ожидаемый результат: []
Результат: Возвращен пустой список.

Проверка строки без разделителей:
Вход: "abc"
Ожидаемый результат: ["abc"]
Результат: Возвращен список с одним элементом.

Проверка строки с разделителем, который не найден:
Вход: "1,2,3" и разделитель ":"
Ожидаемый результат: ["1,2,3"]
Результат: Возвращен список с одним элементом, т.к. разделитель не найден.

4. Метод contains
Проверка строки, которая не содержит символа:
Вход: "SkyPro", символ "X"
Ожидаемый результат: False
Результат: Символ не найден, возвращено False.

Проверка с пустой строкой:
Вход: "", символ "S"
Ожидаемый результат: False
Результат: Пустая строка не может содержать символ, возвращено False.

Проверка с None:
Вход: None, символ "S"
Ожидаемый результат: Исключение TypeError
Результат: Поднято исключение TypeError.

5. Метод delete_symbol
Проверка удаления символа из строки, где символ не найден:
Вход: "SkyPro", символ "X"
Ожидаемый результат: "SkyPro"
Результат: Символ не найден, строка осталась неизменной.

Проверка удаления символа из пустой строки:
Вход: "", символ "X"
Ожидаемый результат: ""
Результат: Пустая строка, ничего не изменилось.

6. Метод starts_with
Проверка строки, которая не начинается с указанного символа:
Вход: "SkyPro", символ "P"
Ожидаемый результат: False
Результат: Строка не начинается с символа, возвращено False.

Проверка с пустой строкой:
Вход: "", символ "S"
Ожидаемый результат: False
Результат: Пустая строка не может начинаться с символа, возвращено False.

7. Метод end_with
Проверка строки, которая не заканчивается на указанный символ:
Вход: "SkyPro", символ "S"
Ожидаемый результат: False
Результат: Строка не заканчивается на символ, возвращено False.

Проверка с пустой строкой:
Вход: "", символ "o"
Ожидаемый результат: False
Результат: Пустая строка не может заканчиваться на символ, возвращено False.

8. Метод is_empty
Проверка строки с содержимым:
Вход: "SkyPro"
Ожидаемый результат: False
Результат: Строка не является пустой, возвращено False.

Проверка строки с пробелами и символами:
Вход: " SkyPro "
Ожидаемый результат: False
Результат: Строка не пустая, возвращено False.

9. Метод list_to_string
Проверка пустого списка:
Вход: []
Ожидаемый результат: ""
Результат: Пустой список, результат — пустая строка.

Проверка списка с одним элементом:
Вход: [1]
Ожидаемый результат: "1"
Результат: Один элемент списка преобразован в строку.

Заключение:
Негативные тесты помогают удостовериться, что методы класса StringUtils корректно обрабатывают нештатные ситуации и возвращают предсказуемые результаты или выбрасывают исключения, если это необходимо. Все проверенные сценарии возвращают ожидаемые результаты, что подтверждает стабильность работы функций при нетипичных входных данных.
